# Skippy C

Skippy C is an interpretted language built with C#

## Running Skippy C

> cd ~/Skippy-C/SkippyC
>
> dotnet restore
>
> dotnet build
>
> dotnet run

## Testing Skippy C

> cd ~/Skippy-C/SkippyCTests
>
> dotnet restore
>
> dotnet build
>
> dotnet test
